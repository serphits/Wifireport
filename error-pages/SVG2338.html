SVG2338

Arris SVG2338 DNS Resolution Error

Arris DNS Resolution Error BGW-Series
95% fix rate

Cause

DNS-over-HTTPS provider returned error

Fix Steps

  1. Open router admin panel and navigate to DNS settings
  2. Change primary DNS to 8.8.8.8 (Google) or 1.1.1.1 (Cloudflare)
  3. Set secondary DNS to 8.8.4.4 or 1.0.0.1
  4. Flush DNS cache: ipconfig /flushdns (Windows) or sudo dscacheutil -flushcache (Mac)
  5. Restart router and connected devices
  6. Test DNS resolution: nslookup google.com
