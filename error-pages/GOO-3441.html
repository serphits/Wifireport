Google GOO-3441 Mesh Network Error

Google Mesh Network Error OnHub
90% fix rate

Cause

Satellite firmware version mismatch with router

Fix Steps

  1. Restart all mesh nodes starting with the main router
  2. Ensure satellite nodes are within optimal range (30-50 feet)
  3. Check all nodes have same firmware version
  4. Re-sync satellite nodes through the router's mesh management page
  5. Avoid placing satellites behind thick concrete walls
  6. Consider wired ethernet backhaul for best performance
