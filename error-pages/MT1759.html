MT1759

Motorola MT1759 Connection Error

Motorola Connection Error MG-Series
88% fix rate

Cause

DHCP server not responding

Fix Steps

  1. Power cycle your router by unplugging for 30 seconds
  2. Verify WAN cable is securely connected to router
  3. Log into router admin panel at 192.168.1.1
  4. Navigate to WAN/Internet settings and verify connection type
  5. Re-enter your ISP credentials if using PPPoE
  6. Check ISP service status at your provider's website
